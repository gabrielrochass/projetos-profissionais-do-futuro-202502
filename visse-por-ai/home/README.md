# Visse por Aí

## Overview
"Visse por Aí" is a web project dedicated to showcasing the vibrant culture, history, and attractions of Recife, Brazil. The website serves as a comprehensive guide for both locals and tourists, providing information about various points of interest, cultural insights, and local characters.

## Project Structure
The project is organized into the following directories and files:

```
visse-por-ai
├── src
│   ├── index.html          # Main entry point for the website
│   ├── home.html           # Home page content
│   ├── components           # Contains reusable components
│   │   ├── header.html      # Header component with navigation
│   │   └── footer.html      # Footer component with copyright info
│   ├── css                 # Stylesheets
│   │   ├── reset.css       # CSS reset for consistent styling
│   │   ├── home.css        # Styles specific to the home page
│   │   └── responsive.css   # Responsive styles for various devices
│   ├── js                  # JavaScript files
│   │   └── main.js         # Main JavaScript for interactivity
│   └── fonts               # Custom fonts
│       └── ADLaMDisplay-Regular.ttf
├── public
│   └── assets              # Public assets
│       ├── images          # Image files used in the website
│       └── logos           # Logo files used in the website
├── .gitignore              # Files to ignore in version control
├── package.json            # Project metadata and dependencies
└── README.md               # Project documentation
```

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd visse-por-ai
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the project.

## Usage
- The home page (`home.html`) provides an introduction to Recife and its attractions.
- The header and footer components are included in the main HTML files for consistent navigation and branding.
- CSS files ensure the website is visually appealing and responsive across different devices.
